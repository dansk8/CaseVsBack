# processmap_case

## Backend (.NET + PostgreSQL)
- Configure a connection string no `appsettings.json`
- Rode migrations:
  - `dotnet ef database update`
- Rode a API:
  - `dotnet run`

Swagger: https://localhost:57242/swagger

## Frontend (React)
- `cd ProcessMap.Web`
- `npm install`
- `npm run dev`

Front: http://localhost:5173
